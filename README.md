# espero os
A simple operating system from scratch.

# Tools
- OS: Void linux
- Compiler: gcc
- Editor: neovim

# Prerequisites
- make
- virtualbox

# Build
```
git clone https://github.com/LinArcX/esperos
cd esperos
mkdir -p output
make
```

# Run esperos in virtualbox
`make run`

# Clean output directory
`make clean_all`
